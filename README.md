# ti-components

A customized style UI components library based on semantic-ui.
This project uses [storybook](https://github.com/storybookjs/storybook) to organize, describe and preview components.

## Setup

## Prerequisites

- node: >=10.15.3
- semantic-ui-react
- storybook
- TypeScript
- styled-components

### Install

```bash
npm install
```

or (with yarn)

```bash
yarn
```

### Run Storybook

```bash
npm run storybook
```
